## 🐵 My Current Stack

- 🏗 Built with [NextJS](https://nextjs.org)
- 🎨 Custom themes powered by [TailwindCSS](https://tailwindcss.org)
- 🧰 Unstyled and accessible UI components with [Headless-UI](https://headlessui.dev)
- 🌗 Dark mode featured by [next-theme](https://github.com/pacocoursey/next-themes)
- 🎣 Performant, flexible and extensible forms with [React Hook Form](https://react-hook-form.com/)
- 🐻 UI State management with [Zustand](https://zustand.surge.sh/)
- 🌸 Data management and Sync with [React-Query](https://react-query.tanstack.com/)
- 🥸 API Mocking with [mswjs](https://mswjs.io/)
- 🤖 SEO support with [next-seo](https://github.com/garmeeh/next-seo)
- 🔒 JWT Authentication with [next-auth](https://next-auth.js.org)
- 🧩 Uncontextualized component visualization with [Storybook](https://next-auth.js.org)
- 🚨 Testing powered by [jest](https://jestjs.io/docs/getting-started)
- 🐙 Component Testing powered by [React Testing Library](https://testing-library.com)
- 🐳 Container-based deployment host powered by [Docker](https://www.docker.com)
